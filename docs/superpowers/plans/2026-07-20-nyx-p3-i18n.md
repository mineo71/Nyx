# Nyx P3 — i18n (English + Ukrainian) Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Steps use `- [ ]`.

**Goal:** Localize all surfaces; ship EN + UK; follow system language with a Settings override; language change reloads windows live.

**Architecture:** Pure `resolveLocale` (tested). Bundled `strings.js` (`window.NYX_STRINGS`) + `i18n.js` runtime exposing `window.t` and auto-applying `[data-i18n]`. Main resolves locale, passes `?lang=` per window (like accent), reloads windows on change. Settings gains a Language control.

**Tech Stack:** vanilla JS, Vitest, Tailwind (already built).

- [ ] **Step 3: Implement `src/core/i18n.js`**
```js
const LOCALES = ['en', 'uk'];

function resolveLocale(setting, systemLocale) {
  if (setting === 'en' || setting === 'uk') return setting;
  if (setting === 'auto' || setting == null || !LOCALES.includes(setting)) {
    const sys = String(systemLocale || '').toLowerCase();
    return sys.startsWith('uk') ? 'uk' : 'en';
  }
  return 'en';
}

module.exports = { resolveLocale, LOCALES };
```

- [ ] **Step 2: Create `src/renderer/i18n/i18n.js`**
```js
// Reads ?lang=, exposes window.t(key), and applies [data-i18n]/[data-i18n-html] on load.
(function () {
  const lang = new URLSearchParams(location.search).get('lang') || 'en';
  const dict = (window.NYX_STRINGS && (window.NYX_STRINGS[lang] || window.NYX_STRINGS.en)) || {};
  window.NYX_LANG = lang;
  window.t = (key) => (key in dict ? dict[key] : key);
  function apply() {
    document.querySelectorAll('[data-i18n]').forEach((el) => { el.textContent = window.t(el.getAttribute('data-i18n')); });
    document.querySelectorAll('[data-i18n-html]').forEach((el) => { el.innerHTML = window.t(el.getAttribute('data-i18n-html')); });
  }
  if (document.readyState === 'loading') document.addEventListener('DOMContentLoaded', apply);
  else apply();
})();
```
