What it does

A quiet bedside companion for the nights you drift off mid-film.

🌙

On-device detection

MediaPipe watches for closing eyes and a dropping head — a windowed, hysteresis model so blinks don't fool it.

🪜

Gentle escalation

Quiet nudge → pause playback → louder nudge → sleep the Mac. Any movement or open eyes cancels it.

🎬

Knows what's playing

Arms when a video's in sight, shows what's on, and saves the title + link you dozed off to.

🔦

Camera-frugal

The webcam only wakes when armed — brief checks while watching, continuous only when confirming a doze.

🔒

Private & offline

No cloud, no accounts, no images stored. Frames are analysed in memory and discarded.

🌍

English & Українська

Follows your system language. Native menu-bar popover and a full dashboard.

See it

Popover, onboarding, settings, and the low-glare nudge overlay.

How it works

Small, pure, unit-tested detection modules; thin macOS glue.

video in sight ──▶ arm camera ──▶ MediaPipe eye-blink + head-pose │ windowed drowsiness model (hysteresis + PERCLOS) │ eyes closed long enough ──▶ escalation ladder nudge → pause → louder nudge → sleep Mac │ any input / eyes open ──▶ cancel
